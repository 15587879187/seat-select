# SeatSelect (vue移动端座位选择)

> [点击这里demo预览](https://www.xollipop.top/ "请使用手机访问")
>
> 此次项目是基于vue编写的类似淘票票和猫眼的电影`移动端`锁座页面,其中包含功能
>
> - 座位图生成
> - 座位预览图生成
> - 座位`留空`检测
> - 座位快速选择`最优座位`算法
> - 自适应影厅大小
> - 座位图左侧导航栏的过道检测
> - 普通座位的选择逻辑
> - 情侣座位的选择逻辑

## Project setup(初始项目)
```
npm install
```

### Compiles and hot-reloads for development (运行开发环境)
```
npm run serve
```

### Compiles and minifies for production(打包)
```
npm run build
```
